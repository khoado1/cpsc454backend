Enable access control (authentication):

Stop MongoDB: brew services stop mongodb/brew/mongodb-community
Start without auth: mongod --dbpath /usr/local/var/mongodb --fork --logpath /usr/local/var/log/mongodb/mongo.log
Connect: mongosh
Create admin user:

use admin
db.createUser({user: "admin", pwd: "password", roles: ["userAdminAnyDatabase", "readWriteAnyDatabase"]})

Stop, then start with auth: mongod --auth --dbpath /usr/local/var/mongodb --fork --logpath /usr/local/var/log/mongodb/mongo.log
Connect with auth: mongosh -u admin -p password
Run as non-root user:

Create a mongodb user: sudo dscl . -create /Users/mongodb
Change ownership: sudo chown -R mongodb /usr/local/var/mongodb
Run as that user: sudo -u mongodb mongod --auth --dbpath /usr/local/var/mongodb --logpath /usr/local/var/log/mongodb/mongo.log --fork
